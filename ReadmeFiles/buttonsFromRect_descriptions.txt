Functions and Variables in buttonsFromRect.py:

1. create_button
   - Description: Creates a button with optional text and an image, and returns its rectangle and associated action.
   - Parameters:
     - text (str): The text to display on the button.
     - x (int): The x-coordinate of the button.
     - y (int): The y-coordinate of the button.
     - width (int): The width of the button.
     - height (int): The height of the button.
     - text_color (Tuple[int, int, int]): The color of the text.
     - surface (pygame.Surface): The surface to draw the button on.
     - action (Callable, optional): The function to execute when the button is clicked.
     - image (pygame.Surface, optional): An optional image to display on the button.
   - Returns: A tuple containing the button's rectangle (pygame.Rect) and the associated action.

2. calculate_button_positions
   - Description: Calculates and returns the positions of various buttons relative to the screen dimensions.
   - Parameters: None.
   - Returns: A tuple containing the calculated positions:
     - button_y (int): The y-coordinate for buttons.
     - left_button_x (int): The x-coordinate for the left button.
     - right_button_x (int): The x-coordinate for the right button.
     - next_button_x (int): The x-coordinate for the "next" button.
     - next_button_y (int): The y-coordinate for the "next" button.
     - back_button_x (int): The x-coordinate for the "back" button.
     - pick_save_x (int): The x-coordinate for the "pick save" button.
     - pick_save_y (int): The y-coordinate for the "pick save" button.
