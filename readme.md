# Initial Setup
1. Install python
2. Setup the virtual environment by navigating to the PokeMii directory and running ```python -m venv venv```
3. ```.\venv\Scripts\activate```
4. ```pip install -r requirements.txt```

And you're all set! Run the game using ```python mainMenu.py``` (while the virtal environment is active of course!)

Tested on python 3.8.5 and 3.13.2
